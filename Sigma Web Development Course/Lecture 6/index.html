Core Web Vitals

The 3 Main Core Web Vitals:

  • LCP (Largest Contentful Paint): How fast the main content loads Goal: Under 2.5 seconds
  • FID (First Input Delay): How quickly the page responds to clicks Goal: Under 100 milliseconds
  • CLS (Cumulative Layout Shift): How stable the page is while loading Goal: Under 0.1
