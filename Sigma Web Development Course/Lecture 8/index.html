What is Block Elments:

Block elements take up the full width available, with a new line before and after them. Examples include <div>, <p>, and <h1>.

What is Inline Elements:

Inline elements only take up as much width as necessary, and do not start on a new line. Examples include <span>, <a>, and <strong>.
