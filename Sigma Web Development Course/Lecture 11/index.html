What is Semantic Tags?

Semantic tags browser ko aur web crawlers ko content samajhne mein madad karte hain.

Aur Ap ke Website ka purpose kia hai aur konsa tag kis purpose ke le use hua hai.

Semantic tags ka istemal karke, aap apne HTML document ki structure aur meaning ko behtar bana sakte hain.

Aur ye SEO (Search Engine Optimization) ke liye bhi faida mand hai.

Semantic Tags

  • <header> - Page ka header section
  • <nav> - Navigation links ke liye
  • <main> - Page ka main content
  • <article> - Independent content piece
  • <section> - Content ka section
  • <aside> - Sidebar ya supplementary content
  • <footer> - Page ka footer section
  • <figure> - Images ya illustrations ke liye
  • <figcaption> - Figure ka caption
  • <time> - Date aur time information

Why they are used?

Semantic tags ka istemal karne se HTML document ki readability aur maintainability behtar hoti hai.

Ye tags web crawlers ko content ko samajhne mein madad karte hain, jo SEO ke liye bhi faida mand hai.

They are used to enhance the accessibility and SEO of the web page.
