What are HTML Entities?

HTML entities are special characters that are used to represent certain symbols or characters in HTML. They are often used to display characters that have a special meaning in HTML, such as <, >, and &.

Common HTML Entities

  • & - Ampersand (&)
  • < - Less than (<)
  • > - Greater than (>)
  • " - Double quote (")
  • ' - Single quote (')
  •   - Non-breaking space
  • © - Copyright symbol (Â©)

Usage of HTML Entities

HTML entities are used to ensure that the HTML code is correctly interpreted by web browsers. For example, if you want to display a less than sign (<) in your HTML content, you would use the entity < instead of the actual character. This prevents the browser from interpreting it as the start of an HTML tag.
